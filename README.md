# Sección de conexión - Azure

Portal de redirección/callback autorizado para conexión con Azure. Incluye los módulos: Contabilidad, Auditorías y Legales.

## Despliegue en Vercel

1. Sube este repositorio a tu cuenta de GitHub/GitLab.
2. En [vercel.com](https://vercel.com) → **Add New** → **Project** → importa el repo.
3. Deploy. La URL de redirección para Azure será la que te asigne Vercel (ej: `https://tu-proyecto.vercel.app`).

## Uso local

Abre `index.html` en el navegador o sirve la carpeta con cualquier servidor estático.
